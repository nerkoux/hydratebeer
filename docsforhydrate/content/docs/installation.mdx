# Installation

## Prerequisites

HydrateBeer requires:
- React 18.0.0 or higher
- Node.js 16 or higher
- A free Tinybird account ([Sign up here](https://tinybird.co))

## Step 1: Initialize HydrateBeer

The CLI handles everything for you:

```bash
npx hydrate-beer init
```

This command will:

1. **Detect your project** (React or Next.js)
2. **Generate a unique project ID** (e.g., `proj_a1b2c3d4e5f6...`)
3. **Prompt for security password** (min 6 characters)
4. **Request Tinybird API token** (get it from [Tinybird Tokens](https://ui.tinybird.co/tokens))
5. **Create configuration files**:
   - `hydrate-beer.config.ts` (with projectId, passwordHash, tinybirdToken)
   - `.env.local` (environment variables)
6. **Auto-install SDK** (`hydrate-beer-sdk`)

### Getting Your Tinybird Token

1. Create a FREE account at [tinybird.co](https://tinybird.co)
2. Go to [Tokens page](https://ui.tinybird.co/tokens)
3. Copy your API token
4. Paste when prompted by `init` command

## Step 2: Deploy Tinybird Schema

Set up your analytics backend:

```bash
npx hydrate-beer setup-tinybird
```

This creates in your Tinybird account:
- **`events` datasource** - Stores all performance metrics
- **`overview` pipe** - Aggregates total stats
- **`realtime_metrics` pipe** - Recent events feed

## Step 3: Install SDK (Manual - if needed)

The CLI auto-installs, but if needed:

**Using npm:**
```bash
npm install hydrate-beer-sdk
```

**Using yarn:**
```bash
yarn add hydrate-beer-sdk
```

**Using pnpm:**
```bash
pnpm add hydrate-beer-sdk
```

**Using bun:**
```bash
bun add hydrate-beer-sdk
```

## Configuration Files

After initialization, you'll have:

### `hydrate-beer.config.ts`
```typescript
export default {
  projectId: "proj_abc123...",
  passwordHash: "$2a$10$...", // bcrypt hash
  tinybirdToken: "p.ey...", // Your Tinybird token
  tinybirdRegion: "us-east", // or eu-central
  sampleRate: 1.0,
  slowRenderThreshold: 16,
  flushInterval: 5000,
  batchSize: 50,
};
```

### `.env.local`
```bash
NEXT_PUBLIC_HYDRATE_BEER_PROJECT_ID=proj_abc123...
NEXT_PUBLIC_HYDRATE_BEER_TINYBIRD_TOKEN=p.ey...
```

## CLI Commands Reference

### `init`
Initialize HydrateBeer in your project
```bash
npx hydrate-beer init
```

### `setup-tinybird`
Deploy Tinybird datasources and pipes
```bash
npx hydrate-beer setup-tinybird
```

### `monitor`
Open live monitoring dashboard (password required)
```bash
npx hydrate-beer monitor
```

## Troubleshooting

### "No configuration found"
Run `npx hydrate-beer init` first.

### "Tinybird API error"
- Check your token is valid
- Verify your region is correct (`us-east` or `eu-central`)
- Ensure you have internet connection

### "Password incorrect"
The password is hashed and stored in `hydrate-beer.config.ts`. If forgotten, re-run `init`.
